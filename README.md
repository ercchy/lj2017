How to use this static repo
===========================

1. Checkout the git

2. Get dependencies

```
npm install
bower install
curl -sf https://www.getlektor.com/install.sh | sh
```

3. Build SCSS

```
gulp
```

4. Run lektor

```
lektor server
```

5. Visit: http://localhost:500

6. Contribute to content in `template/contents.html`
